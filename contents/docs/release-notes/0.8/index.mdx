## Install

```bash
npm install @rocicorp/zero@0.8
```

## Features

- **Result Type.** You can now tell whether a query is complete or partial. ([docs](../react))
- **OTEL Tracing.** Initial tracing support. (see [`ZERO_LOG_TRACE_COLLECTOR`](../zero-cache-config))
- **TIMESTAMPZ.** Add support for `TIMESTAMPZ` column type. ([docs](TODO))
- **`ssl=<mode>`**. You can disable `tls` when `zero-cache` connects to DB with `ssl=disable` or `sslmode=disable`. ([docs](TODO))
- **enum support**. Enums are now supported in Postgres schemas. They are replicated to the client as strings. ([docs](../postgres-support#column-types))

## Breaking Changes

See [hello-zero](https://github.com/rocicorp/hello-zero/commit/a41d95d995a3eed1ba17e2822deac79dccfe2b6c) for example update.

- `useQuery` in React now returns a 2-tuple of `[rows, result]` where `result` is an object with a `type` field.
- `postProposedMutation` in write permissions for `update` renamed to `postMutation` for consistency.
- `TableScheamToRow` renamed to `Row` to not be so silly long.
